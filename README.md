# 👗 Bisma Ladies Tailor

A modern and responsive website for **Bisma Ladies Tailor**, designed to showcase tailoring services with an elegant and user-friendly interface. The website provides customers with information about services, pricing, contact details, and makes it easy to connect with the tailor.

## 🌐 Live Demo

🔗 https://bisma-ladies-tailor.netlify.app/

---

## ✨ Features

- 🎨 Modern and elegant design
- 📱 Fully responsive layout
- 👗 Tailoring services section
- 🖼️ Beautiful image gallery
- 📞 Contact information
- ⚡ Fast loading performance
- 🌍 Cross-browser compatible

---

## 🛠️ Built With

- HTML5
- CSS3

---

## 🎯 Purpose

This project was created as a **frontend practice project** to improve web development skills while building a professional business website.
